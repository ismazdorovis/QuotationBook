# Проект «Цитатник»

Сервис для отображения случайных цитат, реализованный в корпоративном стиле «Газпромнефти».

## 🏗 Архитектура
Проект построен на базе микросервисного подхода с использованием Docker:
- **Web-сервер**: Nginx (проксирует запросы к бэкенду).
- **Backend**: Node.js (Express) — API для выдачи случайных цитат с системой авторизации.
- **Кэширование**: Redis — используется для мгновенной отдачи цитат (Cache-Aside pattern), снижая нагрузку на БД.
- **База данных**: PostgreSQL 15 — надежное хранение текстов цитат и авторов.

## 🚀 Быстрый старт

### Требования
- Docker
- Docker Compose

### Запуск
1. Клонируйте репозиторий в локальную папку.
2. Перейдите в директорию проекта:
   ```bash
   cd QuotationBook
   ```
3. Запустите сборку и старт контейнеров:
   ```bash
   docker compose up --build -d
   ```
4. Откройте браузер и перейдите по адресу: `http://localhost`

## 🔐 Административная панель
Доступ к управлению цитатами осуществляется по адресу: `http://localhost/admin.html`.
Доступ защищен механизмом Basic Auth.

## ⚙️ Конфигурация
Все настройки хранятся в `docker-compose.yml`. 
База данных инициализируется с помощью файла `init.sql` при первом запуске.
